Grooming Service

Business owner web app

Challenge

BYP grooming service was built years ago but not yet launched to customers. Add service flow is complicated and difficult for the user to add or modify service. Book appointment flow cannot let users book service with multiple add-ons. Some other concerns are unable to apply promotions, customers not able to add more than one pet, etc.

Solution

The goal is to simplify the current flow and make sure easy to use. We removed the location feature as it is, and make it global. merged punch pass and promotion together and removed the market tab for now. Removed booking quantity and team members' capacity, and add a new team member's availability tab.
