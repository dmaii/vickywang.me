Challenge

BYP grooming service was built years ago but not yet launched to customers. Add service flow is complicated and difficult for the user to add or modify service. Book appointment flow cannot let users book service with multiple add-ons. Some other concerns are unable to apply promotions, customers not able to add more than one pet, etc.

Solution

The goal is to simplify the current flow and make sure easy to use. We removed the location feature as it is, and make it global. merged punch pass and promotion together and removed the market tab for now. Removed booking quantity and team members' capacity, and add a new team member's availability tab.

Insights

Adjustable price & duration

Booking Flexibility
  • Booking is complicated, has large variations, price duration needs to adjust
  • Different pet sizes, coats, and behavior inspect the price
  • Pricing based on pet size, pet coat, behavior, adjust duration and time at check-in all the time

Overlap booking

Team member availability
  • Business owner managing availability
  • Allow team members to see other members' availability
  • Not allow customers to pick the team member, assign team member a few days before the appointment

Booking complexity

Consultation/Appointment approval
  • Customers don't know which package or service they should buy and need to confirm an appointment/consultation first
  • Contact customer if the booking information need to be adjusted

Appointment preparation

Set buffer between appointments
  • The buffer between each appointment is necessary, they may need to clean up or prepare things for the next customer

Time slots

Avoid block groomers' availability
  • Customers don't know which package or service they should buy and need to confirm an appointment/consultation first

No show

notification & alet
